Capítulo 7: Ejercicios

< Ejercicio 7.1 />

Montero Gómez Evelin

Cuestionario

  • a) Todas las secuencias de comandos pueden escribirse en términos de tres tipos de estructuras de control: secuencia , selección y repeticion.

  • b) La instrucción de selección doble if...else se utiliza para ejecutar una acción cuando una condición es verdadera, y otra acción cuando esa condición es falsa.

  • c) Al proceso de repartir un conjunto de instrucciones un número especifico de veces se le llama repetición controlada por contador.

  • d) Cuando no se sabe de antemano cuántas veces se repetirá un conjunto de instrucciones, se puede usar un valor centinela (o uno de señal , de bandera o de prueba ) para terminar la repetición.
