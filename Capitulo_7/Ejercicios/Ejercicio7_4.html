Escribir instrucciones de código

Escriba instrucciones en JavaScript para realizar cada una de las siguientes tareas:

  • a) Declara la variable suma & x.
    let suma, x;

  • b) Asignar 1 a la variable x.
    x = 1;

  • c) Asignar 0 a la variable suma.
    suma = 0;

  • d) Sumar la variable x a suma y asignar el resultado a la variable suma.
    suma += x;

  • e) Imprimir "La suma es: ", seguido del valor de la variable suma.
    document.writeln("<h4>La suma es: "+ suma+" </h4>");
