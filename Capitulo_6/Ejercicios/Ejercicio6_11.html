¿Qué aparece en el diálogo de alerta cuando se ejecuta cada una de las siguientes instrucciones en JavaScript? Asuma que x = 2 y y = 3.

  1. window.alert("x = " + x);
  2. winsow.alert("El valor de x + x es" + (x + x));
  3. window.alert("x =");
  4. window.alert((x + y) + "=" + (y + x));
