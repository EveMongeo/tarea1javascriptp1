Montero Gómez Evelin

¿Qué hace la siguiente función?

// El parámetro b debe ser entero
// positivo para evitar recursividad infinita
function misterio( a, b )
{
 if ( b == 1 )
  return a;
 else
  return a + misterio(a, b - 1);
}

Lo que hace la función es sumar el valor de "a" a un total de números de veces de "b".
