Primo entero

Se dice que un entero es primo si es mayor que 1 y se puede dividir sólo entre 1 y sí mismo. Por ejemplo, 2, 3, 5 y 7 son primos pero 4, 6, 8 y 9 no.

  1. Escriba una función que determine si un número es primo.

    2. function esPrimo(numero){
     if (numero == 0 || numero == 1 || numero == 4)
      return false;
     for (let x = 2; x < numero / 2; x++) {
      if (numero % x == 0) return false;
     }
     return true;
    }

  2. Use esta función en una secuencia de comandos que determine e imprima todos los números primos entre 1 y 10,000. ¿Cuántos de estos 10,000 números tiene que probar en realidad antes de asegurarse que haya encontrado todos los primos? Muestre los resultados en un elemento <textarea>.

  3. En un principio podríamos pensar que n/2 es el límite superior con el que debemos probar para ver si un número es primo, pero sólo necesitamos llegar hasta la raíz cuadrada de n. ¿Por qué? Vuelva a escribir el programa usando el método Math.sqrt para calcular la raíz cuadrada y ejecútelo de las dos formas. Estime la mejora en el rendimiento.
