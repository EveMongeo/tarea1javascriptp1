Montero Gómez Evelin

Funciones

Implemente las siguientes funciones:

  1. La función Celsius devuelve el equivalente en Celsius de una temperatura en Fahrenheit, mediante el cálculo:
    C = 5.0 / 9.0 * ( F - 32 );

    2. function Celsius(F){
     return (5.0 / 9.0) * (F - 32);
    }

  2. La función Fahrenheit devuelve el equivalente en Fahrenheit de una temperatura en Celsius, mediante el cálculo:
    F = 9.0 / 5.0 * C + 32;

    3. function Fahrenheit(C){
     return (9.0 / 5.0) * (C + 32);
    }

  3. Use esta funciones para escribir una secuencia de comandos que permita al usuario introducir una temperatura en Fahrenheit o Celsius y muestre el equivalente en Celsius o Fahrenheit.
