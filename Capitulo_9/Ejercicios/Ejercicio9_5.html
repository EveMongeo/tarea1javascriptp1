< Ejercicio 9.5 />

Montero Gómez Evelin

Localice el error en cada uno de los siguientes segmentos de programa y explique cómo corregirlo:

  • method g()
    {
      document.writeln( "Dentro del método g" );
    }

    • >> method no es la palabra que se usa para definir una función:
    function g()
    {
      document.writeln( "Dentro del método g" );
    }

  • >> Regresa la suma de sus argumentos
    function suma(x, y)
    {
      var resultado;
      resultado = x + y;
    }

    • >> Está devolviendo la variable resultado sin valor:
    function suma(x, y)
    {
      resultado = x + y;
      var resultado;
    }

  • function f( a );
    {
    document.writeln( a );
    }

    • >> Eliminar el punto y coma que está después del parámetro:
    function f( a )
    {
    document.writeln( a );
    }
